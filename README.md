## üöÄ Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- NPM (Comes with Node.js)

### Installation
1. Clone the repository:
```bash
git clone [repository-url]
```

2. Install dependencies:
```bash
   npm run dev
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
